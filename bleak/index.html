Bleak

A collaboration with JP Guevremont, Bleak is a Chrome extension weather source for those that see the glass as half empty.

Chrome Extension
Role: UX + UI Design
2015

The project

The goal was to design a system that displayed custom statements that were tied to the current weather conditions, plus be reflective of both the season and current temperature. A Google Doc was set up where statements were added to 5 degree temperature increments, and tagged to specific predetermined weather conditions. The intention was to have the statement never seem random, and be reflective of current conditions. Regardless of how nice the weather is, Bleak is all but guaranteed to darken even the brightest day (or night). Updates included temperature unit toggles (Celsius/Fahrenheit), and a 7-day forecast.
